ReGrowtopia (GrowUnity) is one of the successful Growtopia Private Servers! What do we mean by successful? It has the most features compared to other servers and most experienced team.

Our Unity private server is a modified version of the popular multiplayer sandbox game, Growtopia, where players can create and manipulate their virtual worlds. HOSTS GO

1. Download and Install(No Root): Hosts GO APK
2. Open Host GO and Click on "Hosts Editor".
3. Click checkbox on the right, and Click "Download Hosts File".
4. Type the link on bellow: "https://regrowtopia.xyz/android.txt"
5. Click on "Download and Apply".
6. Back and click on "START".
7. Open Growtopia and Connect!

POWERTUNNEL

1. Download and Install": PowerTunnel APK
2. Open Plugin button beside Setting button.
3. Find Hosts and open the Setting.
4. Type the link on bellow: "https://regrowtopia.xyz/android.txt"
5. Back and click on "CONNECT" button.
6. Open Growtopia and Connect!

VIRTUAL HOST

1. Download and Install: Virtual Host APK
2. Download Host: Download Host Here.
3. After install, open Virtual Host and click on "SELECT HOSTS FILE".
4. Find the Host file and click it.
5. Open Growtopia and Connect!

IPHONE

1. Download Surge 5 from App Store: Surge 5 IOS
2. Press "OK".
3. Click on "Default.conf".
4. Click on "IMPORT" part -> Download Profile from URL.
5. Put url "https://regrowtopia.xyz/ios.txt" and click "OK" and "Done"
6. Press "SETUP" and then agree to policy by clicking "OK" and "Allow" for VPN Configuration!.
7. Done, then open "Growtopia" and Connect!

Windows

1. Press Win+R (Windows Key + R).
2. Type in C:\Windows\System32\drivers\etc in the textbox.
3. Find file named "hosts".
4. Right-click on "hosts" file and edit it with Notepad/text editor.
5. Add in the IP:
107.23.190.166 www.growtopia1.com
107.23.190.166 www.growtopia2.com
6. Save the file, then open Growtopia

Mac

Mac

1. Open Finder and, in the Finder menu bar, select Go > Go to Folder.
2. Type /private/etc/hosts in the textbox and press Return.
3. Move hosts file to Desktop.
4. Open the hosts file, and add in the IP:
107.23.190.166 www.growtopia1.com
107.23.190.166 www.growtopia2.com
5. Save the file and return it to original location
